O que é um array?

Arrays são objetos semelhantes a listas que vêm com uma série de métodos embutidos para realizar operações de travessia e mutação. 

          
              const Array = ["elemento indice 0", "elemento indice 1"]

.lenght

utilizado para saber o tamaho maximo de elementos do array

            
              const Array = ["elemento indice 0", "elemento indice 1"];
              console.log(Array.length);

O retorno do console será 2, pois é referente a quantidade de elementos

.pop

O método pop() remove o último elemento de um array e retorna aquele elemento.

              
                const Array = ["elemento indice 0", "elemento indice 1"];
                Array.pop();
                console.log(Array);

o retorno será -> [ 'elemento indice 0' ]

.shift

O método shift() remove o primeiro elemento de um array e retorna aquele elemento.

                
                  const Array = ["elemento indice 0", "elemento indice 1"];
                  Array.shift();
                  console.log(Array);

o retorno será -> [ 'elemento indice 1' ]

.unshift

O método unshift() Adicionar um item no inicio do Array

                  
                    const Array = ["elemento indice 0", "elemento indice 1"];
                    Array.unshift("adicionando elemento ao inicio");
                    console.log(Array);

o retorno será -> [ 'adicionando elemento ao inicio', 'elemento indice 0', 'elemento indice 1' ]

.push

O método push() Adicionar um item ao final do Array

                
                  const Array = ["elemento indice 0", "elemento indice 1"];
                  Array.push("elemento adicionado");
                  console.log(Array);

o retorno será -> [ 'elemento indice 0', 'elemento indice 1', 'elemento adicionado' ], que indica que ele foi adicionado do Array

.splice

O método splice() Remove itens de uma posição de índice

                  
                    const Array = ["elemento indice 0", "elemento indice 1"];
                    Array.splice(0,1); //posição 0 remove 1 item
                    console.log(Array);

o retorno será -> [ 'elemento indice 1' ]

.slice

O método slice() Copia um Array

                  
                    const Array = ["elemento indice 0", "elemento indice 1"];
                    const ArrayNew = Array.slice();
                    console.log(ArrayNew);

o retorno será -> [ 'elemento indice 0', 'elemento indice 1' ]

.map

O método map() invoca a função callback passada por argumento para cada elemento do Array e devolve um novo Array como resultado.

                    
                      const Array = ["elemento índice 0", "elemento índice 1"];
                      const modifiedArray = Array.map(element => element + " modificado");
                      console.log(modifiedArray);

o retorno será -> [ 'elemento índice 0 modificado', 'elemento índice 1 modificado' ]
